# vue + node + koa+ mongodb+vant

#### 项目介绍
**vue + node + koa+ mongodb+vant**
增删改查小功能 前后端分离。本打算做的图书系统,最后懒得做了 原理是一样的 比图书系统功能强大,请自行举一反三


#### 后端安装教程
1. cd server
2. npm install
3. node app.js

#### 前端安装教程
1. cd client
2. npm install
3. npm run dev

#### mongodb安装

1. 创建book数据库
2. 创建user表
3. user表字段：userName,password
